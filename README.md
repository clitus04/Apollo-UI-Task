# APOLL UI TASK

## Run Locally

Clone the project

```bash
  git clone https://github.com/clitus04/Apollo-UI-Task.git
```

Go to the project directory

```bash
  cd Apollo-UI-Task
```

Install dependencies

```bash
  npm install or yarn install
```

Start the server

```bash
  npm run start or yarn dev
```
## Tech Stack

- React
- Vite
- Primereact
- Primeflex
